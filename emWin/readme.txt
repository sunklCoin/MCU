Segger emWin v5.38a for NXP Platforms
=====================================

These emWin library files were built using:

- arm-none-eabi-gcc gcc version 5.4.1
- IAR ANSI C/C++ Compiler 7.80.4 for ARM
- Keil ARM C/C++ Compiler 5.06 update 3 (build 300) [MDK-ARM Professional 5.21a]
- Kinetis Design Studio Version 3.2.0
- MCUXpresso IDE v0.0.0_alpha [Build 291] [2017-02-03]

The libraries are compatible only with application generated using
hardware floating point with hard-ABI.
